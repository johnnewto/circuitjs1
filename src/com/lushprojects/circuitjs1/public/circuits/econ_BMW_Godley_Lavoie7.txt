## Bank-Money World Model (BMW)

**Source:** Godley & Lavoie (2007) Chapter 7  
**Based on:** sfcr R package implementation

This model demonstrates **stock-flow consistent (SFC) modeling** of a monetary economy with banks creating money through lending.

### Key Features

- Banks create money by issuing loans
- Households hold deposits and consume
- Firms invest and employ workers
- No government sector (simplified version)

### Key Variables

| Variable | Description |
|----------|-------------|
| **Y** | National income (output) |
| **C** | Consumption |
| **I** | Investment |
| **K** | Capital stock |
| **Mh** | Household money holdings |
| **Ld** | Loan demand |

### Steady-State Values

At steady state (t → ∞):
- Y = 160, C = 144, I = 16
- K = 160 (capital-output ratio = 1)
- Mh = Ms = Ld = Ls = 160

### References

> Godley, W., & Lavoie, M. (2007). *Monetary Economics: An Integrated Approach to Credit, Money, Income, Production and Wealth*. Palgrave Macmillan.

@parameters s_s_Params
  rl = 0.025  # Interest rate on loans
  \alpha0 = 20  # Autonomous consumption
  \alpha1 = 0.75  # Propensity to consume from income
  \alpha2 = 0.10  # Propensity to consume from wealth
  \delta = 0.1  # Depreciation rate
  \gamma = 0.15  # Adjustment speed of capital to target
  \kappa = 1  # Capital-output ratio target
  pr = 1  # Labor productivity
@end

@equations MarketClearing
  Cs ~ Cd  # Consumption supply
  Is ~ Id  # Investment supply
  Ns ~ Nd  # Labor supply
  Ls ~ integrate(diff(Ld))  # Loan supply
@end

@equations Firms
  Y ~ Cs + Is  # National income (output)
  WBd ~ Y - lag(rl, 1) * lag(Ld, 1) - AF  # Wage bill demanded
  AF ~ \delta * lag(K, 1)  # Amortization funds = delta * lag(K,1)
  Ld ~ integrate(Id - AF)  # Loan demand
  KK ~ t  # hint7
  Y5 ~ lag(KK, 1)  # hint5
  Y6 ~ KK - lag(KK, 1)  # hint6
@end

@equations Households
  YD ~ WBs + lag(rm, 1) * lag(Mh, 1)  # Household disposable income = WBs + lag(rm,1) * lag(Mh,1)
  Mh ~ Mh_init + integrate(YD - Cd)  # Household money holdings (deposits)
@end

@equations Banks
  Ms ~ integrate(diff(Ls))  # Money supply (bank deposits)
  rm ~ rl  # Interest rate on deposits
@end

@equations WageBill
  WBs ~ W * Ns  # Wage bill supplied
  Nd ~ Y / pr  # Labor demand
  W ~ WBd / max(Nd, 0.01)  # Wage rate
@end

@equations Consumption
  Cd ~ \alpha0 + \alpha1 * YD + \alpha2 * lag(Mh, 1)  # Consumption demand
@end

@equations Investment
  K ~ K_init + integrate(Id - DA)  # Capital stock
  DA ~ \delta * lag(K, 1)  # Depreciation actual = delta * lag(K,1)
  KT ~ \kappa * lag(Y, 1)  # Target capital stock = kappa * lag(Y,1)
  Id ~ \gamma * (KT - lag(K, 1)) + DA  # Investment demand
@end

@equations Verify
  check_money ~ Mh - Ms  # Money market
  check_loans ~ Ls - Ld  # Loan market
  check_labor ~ Ns - Nd  # Labor market
  firm_profit ~ Y - WBd - lag(rl, 1) * lag(Ld, 1) - AF
@end
