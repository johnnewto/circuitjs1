# 1-year delay comparison (step + sine)

This debug circuit compares two delay patterns side-by-side:

1) lag-based delay (uses Expr lag() history)
2) stock-based first-order delay (dY/dt = (X-Y)/tau)
3) delay() first-order delay shorthand

Notes:
- `lag(x,1)` in this build uses history + smoothing behavior (not ideal pure transport).
- stock version is a classic first-order exponential delay with time constant `tau`.
- `delay(x,tau)` is shorthand for the same first-order stock dynamics.
- time unit is years, so `1` means 1 year.
@end
