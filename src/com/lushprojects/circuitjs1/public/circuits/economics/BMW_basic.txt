# Model Documentation

## Bank-Money World Model (BMW)
## Flow-Mode Equation Implementation

**Source:** Godley & Lavoie (2007) Chapter 7  
**Based on:** sfcr R package implementation

This model provides an **SFC Table** version of BMW using EquationTable row modes.

### Approach

- **FLOW mode rows** (`A->B`) for directional transactions
- **STOCK mode rows** for state accumulation (`Mh`, `Ld`, `K`)
- **PARAM mode rows** for exogenous/behavioral constants
- **Transaction matrix** for flow tracking

### Key Variables

| Variable | Description |
|----------|-------------|
| **Y** | National income (output) |
| **C** | Consumption |
| **I** | Investment |
| **K** | Capital stock |
| **Mh** | Household money holdings |
| **Ld** | Loan demand |

### Steady-State Values

At steady state (t → ∞):
- Y = 160, C = 144, I = 16
- K = 160 (capital-output ratio = 1)
- Mh = Ms = Ld = Ls = 160

### References

> Godley, W., & Lavoie, M. (2007). *Monetary Economics: An Integrated Approach to Credit, Money, Income, Production and Wealth*. Palgrave Macmillan.
