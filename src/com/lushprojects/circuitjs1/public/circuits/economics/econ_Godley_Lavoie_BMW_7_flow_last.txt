# Model Documentation

## Bank-Money World Model (BMW)
## Flow/Param Equation Implementation using last()

**Source:** Godley & Lavoie (2007) Chapter 7  
**Based on:** sfcr R package implementation  
**Reference:** https://joaomacalos.github.io/sfcr/articles/articles/gl5-bmw.html

This model provides a **flow/param only** version of BMW, using `last(name)`
for all lagged values (`name[-1]` in the sfcr notation).

### Approach

- **FLOW mode rows** (`A->B` or single node) for all behavioral/accounting equations
- **PARAM mode rows** for exogenous/behavioral constants
- `last(name)` replaces all `lag(name, 1)` / `name[-1]` lagged references
- No STOCK or VOLTAGE mode rows

### Key Variables

| Variable | Description |
|----------|-------------|
| **Y** | National income (output) |
| **C** | Consumption |
| **I** | Investment |
| **K** | Capital stock |
| **Mh** | Household money holdings |
| **Ld** | Loan demand |

### sfcr Equations (original R notation)

```r
bmw_eqs <- sfcr_set(
  Cs ~ Cd,
  Is ~ Id,
  Ns ~ Nd,
  Ls ~ Ls[-1] + Ld - Ld[-1],
  Y ~ Cs + Is,
  WBd ~ Y - rl[-1] * Ld[-1] - AF,
  AF ~ delta * K[-1],
  Ld ~ Ld[-1] + Id - AF,
  YD ~ WBs + rm[-1] * Mh[-1],
  Mh ~ Mh[-1] + YD - Cd,
  Ms ~ Ms[-1] + Ls - Ls[-1],
  rm ~ rl,
  WBs ~ W * Ns,
  Nd ~ Y / pr,
  W ~ WBd / Nd,
  Cd ~ alpha0 + alpha1 * YD + alpha2 * Mh[-1],
  K ~ K[-1] + Id - DA,
  DA ~ delta * K[-1],
  KT ~ kappa * Y[-1],
  Id ~ gamma * (KT - K[-1]) + DA
)
```

### Steady-State Values

At steady state (t → ∞):
- Y = 160, Cd = 144, Id = 16
- K = 160 (capital-output ratio = 1)
- Mh = Ms = Ld = Ls = 160

### References

> Godley, W., & Lavoie, M. (2007). *Monetary Economics: An Integrated Approach to Credit, Money, Income, Production and Wealth*. Palgrave Macmillan.
