# Variable Browser - Quick Reference

## Quick Access

- **Menu**: Edit → Variable Browser...
- **Shortcut**: `\` (backslash key)

## What It Does

Shows all available variables in your circuit:
- **Stocks** (from Table elements)
- **Labeled Nodes** (named connection points)
- **Variables** (from equations)

## How To Use

1. Press `\` or use Edit → Variable Browser...
2. Click any variable name
3. Variable appears on canvas as labeled node
4. Drag it to where you want it
5. Dialog stays open - click more variables as needed

## Features

✓ Non-modal - doesn't block the canvas  
✓ Always on top - stays visible  
✓ Auto-refresh - updates when circuit changes  
✓ Click to place - instant variable placement  
✓ Alphabetically sorted - easy to find  

## Tips

- Dialog stays open - no need to reopen for each variable
- Newly placed variables are auto-selected for immediate positioning
- Works great with ODEElm equations
- Variables in the browser can be used in any equation field
- Click Refresh button to manually update the list

## Integration

The Variable Browser uses the same variable sources as:
- ODEElm equation autocomplete
- TableElm cell references
- Other equation-based elements

This ensures consistency across the application.
